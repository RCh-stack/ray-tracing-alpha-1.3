Статистика по пройденным тестам

В системе доступен просмотр статистики по пройденным тестам.

На форме со статистикой информация отображается в следующем виде:
1. ФИО студента.
2. Тема теста.
3. Оценка.
4. Дата прохождения.

При установке настройки просмотра подробной информации, пользователь может выбрать необходимый тест и двойным нажатием по нему перейти в форму просмотра информации по попытке прохождения. На форме отображаются тестовые вопросы, ответы к ним и пользовательские ответы. Если пользователь верно ответил на вопрос, он будет подсвечен зеленым цветом. Если неверно, то пользовательский ответ будет подсвечен красным цветом, а верный ответ - зеленым.
