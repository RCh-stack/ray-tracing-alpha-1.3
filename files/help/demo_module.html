Модуль «Демонстрация»

Модуль предназначен для настройки и выполнения пошаговой реализации алгоритма трассировки лучей.

Основной функционал:

1. Пользовательская настройка параметров для демонстрации.
Включает в себя настройку параметров объектов (расположение, цвет), сцены, источников освещения, форматов вывода поясняющей информации, использования дополнительных возможностей трассировщика (отражения, преломления).

2. Пошаговая демонстрация алгоритма трассировки лучей.

3. Переходы между этапами выполнения алгоритма.

4. Получение информации по визуализации каждого этапа выполнения алгоритма.

5. Сохранение полученных изображений с моделями.
