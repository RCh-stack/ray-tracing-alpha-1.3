Настройки демонстрации

Система предоставляет возможность пользовательской настройки параметров для демонстрации.
Настройки можно использовать по умолчанию, для этого необходимо проставить галочку у пункта «Использовать настройки по умолчанию».

Параметры для сферы включают в себя следующее:
1. Координаты расположения сферы в формате (x, y, z).
2. Цвет сферы (выбирается из списка, в системе задается как и координаты).
3. Радиус сферы.
4. Показатель преломления для сферы.

Параметры для освещения включают в себя следующее:
1. Координаты расположения источника в формате (x, y, z).
2. Показатель интенсивности освещения.

Форматы вывода поясняющей информации возможны следующие:
1. Наименование этапов без пояснений.
2. Краткое описание каждого этапа реализации.
3. Подробное описание этапов с демонстрацией кода на языке C++.

Также возможны следующие настройки:
1. Отображать/не отображать сцену.
2. Использовать/не использовать несколько источников освещения.
3. Использовать/не использовать отражения.
4. Использовать/не использовать преломления.
5. Изменить расположение генерируемого изображения.

Чтобы перейти к демонстрации, необходимо нажать на кнопку «Начать демонстрацию» внизу формы.
