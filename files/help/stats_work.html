Статистика по выполненным работам

Пользователь может просмотреть свою статистику по отправленным лабораторным работам, с помощью модуля «Статистика по работам».

На форме со статистикой информация отображается в следующем виде:
1. ФИО студента.
2. Тема лабораторной работы.
3. Статус сдачи работы.
4. Дата сдачи работы.
