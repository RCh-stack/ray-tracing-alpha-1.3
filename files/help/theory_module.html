Модуль «Теория»

Модуль предназначен для просмотра теоретического материала по поставленным требованиям.

Основной функционал модуля включает в себя просмотра данных из теоретических файлов, переходы между страницами с помощью кнопок «К следующей странице» и «К предыдущей странице», а также с помощью оглавления.

Выбранная текущая страница подсвечивается в содержании (настраивается администратором).
