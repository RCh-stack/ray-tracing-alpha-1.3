Модуль «Тестирование»

Модуль предназначен для прохождения тестирований по имеющимся темам.

Основной функционал:

1. Пользовательская настройка тестирования (тема теста, режим).

2. Прохождение тестов с возможностью перехода между вопросами и с сохранением ранее отмеченных ответов.

3. Выдача подробной информации о завершенном тестировании.

4. Сохранение результатов тестирования с возможностью подробного анализа по каждому вопросу.

5. Просмотр статистики по всем пройденным тестам.
